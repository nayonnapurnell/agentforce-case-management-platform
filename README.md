# Agentforce Case Management Platform
A Salesforce case management system that uses Agentforce to assist with case triage, summarization, routing, and integration workflows.

# Project Overview

An AI-powered Salesforce Case Management platform that:

- Processes customer support cases
- Uses Agentforce for AI summarization/routing
- Integrates with an external system
- Uses asynchronous Apex processing
- Includes an LWC dashboard
- Handles errors and retries
- Scales for high-volume processing
